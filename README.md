# BSP Content Utilities

## Instalation
1. Install Python 3.12 or above.
2. Install libraries using pip via Command-line or Terminal:
```pip install PyQt5```
```pip install valvebsp```

## Default Content Checker.py
Checks if the BSP file contain files from lists of default content.

## Default Content Cleaner.py
Deletes files that exists in lists of default content from selected directory.
